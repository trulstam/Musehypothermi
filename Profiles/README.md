# Profiles directory

Denne mappen inneholder profiler og relaterte hjelpefiler som brukes av GUI og kontroller for testkjøring. Hvis GitHub viser flere filer enn det som er tilgjengelig lokalt (for eksempel `log debug on.txt`), sørg for at lokalt repo er synkronisert mot fjernlageret (`git fetch`/`git pull`) eller legg til filen manuelt.

Loggfiler som `log debug on.txt` kan legges her for referanse, men merk at store loggfiler kan gjøre GUI tregt. Vurder å komprimere eller rotere loggene før de sjekkes inn.
